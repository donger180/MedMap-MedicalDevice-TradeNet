# MED-MAP Market Access Knowledge Hub / MED-MAP 医疗产品市场准入知识库

> **Data-driven market expansion platform for medical products**  
> **数据驱动的医疗产品市场扩展平台**

MED-MAP is a Singapore-based platform helping medical device, IVD, digital health and biopharmaceutical manufacturers accelerate market access, clinical validation, regulatory registration, distributor matching and commercial growth in Southeast Asia.

MED-MAP 是一家总部位于新加坡的平台，帮助医疗器械、IVD、数字医疗及生物医药企业在东南亚完成市场准入、临床验证、产品注册、经销商匹配与商业增长。

## Deployment / 部署

1. Create a GitHub repository, for example `med-map-market-access`.
2. Upload all files in this folder to the repository root.
3. Go to **Settings → Pages**.
4. Select **Deploy from a branch**.
5. Choose branch `main` and folder `/root`.
6. Save.
7. Optional: configure custom domain `www.med-map.org`.

---

## Suggested Custom Domain / 推荐自定义域名

```text
www.med-map.org
```

For GitHub Pages, add a `CNAME` file containing:

```text
www.med-map.org
```

Then configure DNS CNAME to point to:

```text
<github-username>.github.io
```
